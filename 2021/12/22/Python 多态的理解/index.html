Python 多态的理解

发表于2021-12-22|更新于2022-06-29
|阅读量:

多态

1）基本

  • 让具有不同功能的函数可以使用相同的函数名，这样就可以用一个函数名调用不同的内容（功能）的函数

2）特点

  • 只关心对象的实例方法是否相同，不关心对象所属类型；
  • 增加代码调用灵活性，通用性、兼容性强；

3）举例

  • len()方法，传入不同的参数，可以得到不同的结果
